# 项目 node 版本统一操作

- 在执行任何命令前请先执行命令： corepack enable
- 其次执行命令：pnpm node -v 如果输出版本号则正常，如果没有则会自动下载该 node 版本

# 运行项目

- node V20.19.4 以上版本
- pnpm i 安装依赖
- pnpm dev 测试环境本地运行

# 打包项目

- pnpm build 打包项目
- pnpm preview 打包预览
- pnpm pub 打包项目并启动 pm2

# 其它命令

- pnpm r 删除无 git 跟踪文件，重装依赖请 pnpm i
